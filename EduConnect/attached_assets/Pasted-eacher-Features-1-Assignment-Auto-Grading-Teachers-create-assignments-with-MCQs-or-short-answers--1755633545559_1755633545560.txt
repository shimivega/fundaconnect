1. Assignment Auto-Grading

Teachers create assignments with MCQs or short answers.

Learners submit ‚Üí system auto-grades MCQs.

Short answers ‚Üí AI-assisted keyword grading (optional).

API:

POST /api/assignments ‚Üí create assignment

POST /api/assignments/:id/submit ‚Üí learner submission

GET /api/assignments/:id/results ‚Üí scores

2. Teacher Marketplace

Teachers can upload lesson plans, slides, worksheets.

Optional pricing (free/paid).

API:

POST /api/marketplace

GET /api/marketplace

PATCH /api/marketplace/:id

3. Teacher Collaboration Boards

Kanban board per subject/grade.

Columns: To Do, In Progress, Done.

Drag-and-drop tasks.

API:

POST /api/collaboration/boards

PATCH /api/collaboration/tasks/:id

4. Resource Analytics

Track views/downloads of teacher resources.

API: GET /api/resources/analytics/:id ‚Üí {views, downloads, uniqueUsers}
